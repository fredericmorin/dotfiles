# Fred's dotfiles

My config files for zsh, git, vscode, etc.

Use `stow` to manage symlink overlays in `$HOME` folder.

Used on OSX daily.

## Usage

To install

```sh
cd ~
git clone git@github.com:fredericmorin/dotfiles.git
cd ~/dotfiles
make install
```
