# Fred's dotfiles

My config files for zsh, git, vscode, etc.

Used on OSX daily.

## Setup

Run `./firstboot.sh` to install brew and minimum app suite

Run `./setup.sh` to replace/refresh config symlinks. Existing files will be saved to the `backup` folder.
